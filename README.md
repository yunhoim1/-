## How to Customize

1. **Edit content**: Open `index.html` and modify the text, sections, and layout
2. **Change colors**: Open `style.css` and modify the gradient colors in the `body` section
3. **Add images**: Upload image files to the repository and reference them in `index.html`

## Structure

- `index.html` - Main website page
- `style.css` - Website styling
- `README.md` - This file
